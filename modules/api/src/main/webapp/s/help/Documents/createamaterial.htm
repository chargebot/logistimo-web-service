Create a material

To create a material, perform the following steps:

1.  Go to Setup > Materials.

2.  Click Add new materials button.

3.  In the ensuing form, enter all the Material details.

4.  Specify user's Material Information, as briefed below:

•      Enter Material name

•      Optional fields: Can be entered by clicking the + sign near the title of this section (these include Short name, Custom ID, Description, Additional information, Tags, Batch management, Is binary valued? and seasonal).

5.  Material's Pricing Information:

•      Optional fields: Can be entered by clicking the + sign near the title of this section (these include MSRP, Retailer's price and Currency). 

6.  Material's Temperature information:

•      Optional fields: Can be entered by clicking the + sign near the title of this section (such as Temperature monitoring)

7.  Click Save to create the new material (or Cancel if you want to exit.)
