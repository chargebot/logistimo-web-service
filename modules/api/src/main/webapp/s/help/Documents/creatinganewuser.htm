Creating a new user 

To create a user, perform the following steps:

1.  Go to Setup > Users.

2.  Click Add new User button.

3.  In the ensuing form, enter all the user details.

4.  Specify user's Identification Details, as briefed below:

•      User ID should preferably be in lower case with no space. Use the Check availability button to check if this user ID is available.

•      Ensure that the password is as per the specified length.

•      Select a user role. User can take on any of the roles as explained below.

§ Operator: An Operator has the ability to view and transact inventory and orders. But not manage entities, users and materials. He can only access logistimo using mobile and HTML application.

§ Manager: Manager of one or more entities, with ability to view, transact and manage (entity/users/materials)

§ Administrator: Owns and configures the domain, and can create and manage multiple entities, users (including Entity Managers and Operators), and materials.

•      Optional Field: If there is a custom ID for this user, which you wish to map the user to, please enter that.

5.  User's Personal Details:

•      Enter user's First Name.

•      Optional fields: Can be entered by clicking the + sign near the title of this section (these include Last Name, Gender and Age).

6.  User's Contact Information:

•      Enter User's Mobile Number as per the specified requirements mentioned below the field.

•      If the user's role is a manager or administrator Email ID is mandatory.

•      Optional fields: Can be entered by clicking the + sign near the title of this section (these include Land Line Number and Email ID in case the role )

7.  Specify Location Details:

•      Enter Country and State in the appropriate fields.

•      Select the Language and Preferred Time zone.

•      Optional Field: Can be entered by clicking the + sign near the title of this section (these include District/ County,Taluk, village/city, street address and zip/pin code).

8.  Click Save to create the user's new account (or Cancel if you do not want to create the new account).
