Edit an entity.

To edit an entity, perform the following steps:

1.  Go to Setup > Entities.

2.  Select the entity that you wish to edit entities on.

•      You can also search for an entity by entering entity name (including partial names) in the Search field and click Search button.

3.  At the bottom of the entity profile page, click Edit entity button.

4.  Edit the required fields of the entity.

5.  Click Save to retain edited information (or Cancel to discard the edited details).
