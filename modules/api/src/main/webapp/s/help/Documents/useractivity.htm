User activity

To view user activity, perform the following steps:

1.  Go to Reports

2.  Select user activity in the Type search field.

3.  Click the From and To time fields.

•      Select the date range from the drop down calendar.

3.  Start typing the user name in the User search field.

•      A list of users will appear in the drop down list

•      Select the user you wish from the list.

4.  Click the Get Report button

5.  The report will be rendered,

•      Select ‘Graphics’ to view of the report in the form of a graph or to view in tabular form select ‘Tables’

•      The report shows the number of logins by the specified user.

•      To view by the number of inventory transactions entered by the user, orders placed and revenues booked, select the appropriate fields.
