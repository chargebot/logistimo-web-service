Filter by material

 To view stock levels by material, perform the following steps:

1.  Go to Inventory > Stock views

2.  Start typing the material name in the Material search field.

•      A list of materials will appear in the drop down list

•      Select the material you wish from the list.

3.  Click Go.

4.  View the material’s current stock

•      The current stock is colour coded as per the status (zero stock, <min stock and >max stock)

•      If batch management is enabled for the material you have selected, an icon like this  is shown in the column beside current stock, click to view batch details.

5.  The min and max levels are also visible.

6.  If manual consumption rate is available (enabled through configuration) then availability of stock over days, weeks or months is shown.

7.  The last updated view is also available

8.  If entity tags are shown, you can also browse stock levels by entity tags.
