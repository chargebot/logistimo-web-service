Edit items in order

To edit items in order, perform the following steps:

1.  Go to Orders > Orders

2.  Click order profile you wish to view.

3.  If you wish to edit items in an order in the ensuing order panel, click ‘Change’ link just above the items list.

4.  Edit quantities of relevant items

5.  If a material was batch enabled then ‘enter by batch’ link is shown, click on link to allocate quantities by batch.

6.  Enter comment (if any)

7.  Click Save to update edited items (or Cancel to exit)

8.  The order quantities will be updated, and changed quantities are highlighted with a blue box around them.
