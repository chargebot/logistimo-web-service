Changing a user's password

To change an existing user's password, perform the following steps:

1.  Go to Setup > Users.

2.  Click on the user ID you wish to edit.

•      You can also search for a user by entering his first name (including partial names) in the Search field and click Search button.

3.  At the bottom of the user's profile page, click Change password button.

4.  Enter the user's Old password, New password (specified password length should be in the range: 4-20)

5.  Reconfirm the new password.

6.  Click Change to set new password (or Cancel to exit the form).
