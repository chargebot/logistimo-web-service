View orders

To view orders, perform the following steps:

1.  Go to Orders > Orders

2.  Lists of orders in the domain are shown (order number, user created, quantity, created date).

3.  Click on any given order to view the order panel.

•      The order details are shown in the order panel (which includes Order status, its metadata and materials)

4.  The orders can be paginated across list by clicking on ‘next’ or ‘prev’ links.

5.  If entity tags are shown, you can also browse by entity tags.

6.  If order is geo coded, a map icon like this  is shown at the top right of the order panel, move your mouse on the icon and the location where the order was taken is shown on the map. Click to expand and view map details.
