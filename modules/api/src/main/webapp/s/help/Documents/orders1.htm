Orders

To view and manage order configuration, perform the following steps:

1.  Go to Configuration > Orders

2.  Specify the general orders, as briefed below:

•      Select whether the purchase order is to be generated or not.

•      Select the automatic posting of issues or receipts transactions when order is shipped and/or fulfilled.

•      Set the transporter details as you wish.

•      Select whether you wish to allow empty orders and whether you wish to mark an order as fulfilled.

•      If any, enter payment options, package size and vendor.

3.  Specify the export of order transactions, as briefed below:

•      Select enable export of orders in a spreadsheet, which is sent as an attachment to the specified Email ID.

•      Select the particular time.

•      Enter the specific Email ID you wish.

4.  Specify the demand visibility for orders, as briefed below:

•      Enter the demand board visibility, whether it should be viewed publicly by all or only private.

•      Customize the banner, header and/or footer by copying the HTML link as desired which will connect you to the image.

5.  Click Save
