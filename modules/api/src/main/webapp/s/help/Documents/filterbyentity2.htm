Filter by Entity

To view orders by entity, perform the following steps:

1.  Go to Orders > Orders

2.  Start typing the entity name in the Entity search field.

•      A list of entities will appear in the drop down list

•      Select the entity you wish from the list.

3.  Click View, to view the entity’s current orders

4.  Click on any given order to view the order panel.

•      The order details are shown in the order panel (which includes Order status, its metadata and materials)
