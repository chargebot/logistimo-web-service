Enter new transactions

To create new transactions, perform the following steps:

1.  Go to Inventory > Transactions

2.  Start typing the entity name in the Entity search field.

•      A list of entities will appear in the drop down list

•      Select the entity you wish from the list.

3.  Start typing the operation name in the Operation search field.

•      A list of operations will appear in the drop down list

•      Select the operation you wish from the list, such as:

i. Issues: they are transactions issued by vendor to customer.

ii.          Receipts: transactions issued by customer to vendor for receipt of stock

iii.         Stock count: transaction containing the number of inventory available or physical stock.

iv.         Discards: the transaction with discarded items.

v.         Transfer: transactions transferred to entity.

4.  Click Add button beside operation Search bar

5.  In ensuing page,

•      If the reason field is shown, select the reason.

•      As per the operation - for issues, discards or transfer select customer if necessary and for receipts or stock count select vendor if necessary.

•      Enter quantity in the quantity fields.

6.  For batch managed materials do the following

•      If option is issue, discards or transfer click enter by batch link and fill the quantity fields starting with the first batch and click Save to save the entries.

•      If the option is receipt or stock count in the ensuing form enter the specific quantity details (manufacturing date optional)

7.  Click Submit button below the page, once you fill the form as required or Reset if you do not wish to make the changes you made in the form.

8.  If material tags are shown, you can also browse by material tags.
