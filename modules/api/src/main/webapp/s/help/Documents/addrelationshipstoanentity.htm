Add Relationships to an entity 

To add relationships to an existing entity, perform the following steps:

1. Go to Setup > Entities.

5.  Select the entity that you wish to add relationships on.

•      You can also search for an entity by entering entity name (including partial names) in the Search field and click Search button.

6.  In the ensuring entity profile page, click Relationships at the top-right.

7.  In the ensuing material listing page, click the Add relationships button.

8.  Select relationship type (Customer or Vendor)

9.  Optional fileds: these include Description and Credit limit

10. Select an entity by checking the box adjacent to the entity you wish.

11. Click Save to add a relationship or Cancel to exit.
