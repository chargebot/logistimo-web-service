Create a new custom report

To create a custom report, perform the following steps:

1.  Go to Configuration > Custom reports.

2.  Click Add template button.

3.  In the ensuing form, enter all the report details.

4.  Specify Report template details, as briefed below:

•      To select the report template that needs to be uploaded, click the Choose File button in the field name field.

•      Enter a report template name and an optional description for the template.

5.  Specify Sheet data, as briefed below:

•      Enter the name of the sheet in the excel report into which the exported Logistimo data should be pasted, data should be entered for at least one of the sheets.

•      Example: enter ‘Inventory’ in the inventory sheet name field, if there is a sheet called ‘Inventory’ into which the inventory data should be pasted.

•      This includes details of Inventory, Users, Entities, Materials, Orders, Transactions, Transaction counts, Inventory trends and Historical inventory snapshot.

•      Correctly name the sheet as it should match with the sheet name in the excel workbook that is uploaded.

6.  Specify Report generation and email section, as briefed below:

•      Select one of the options for frequency of report generation.

i. If ‘Daily’ is selected, then enter the time at which the report should be generated.

ii.          The time should be entered in UTC time in hh:mm format.

iii.         If "Weekly" is selected, select the day of the week on which the report should be generated.

iv.         If ‘Monthly’ is selected, the report is generated on the 1st day of every month.

•      In the "Email to" section, select the list of administrators to whom the report should be emailed.

7.  Click Save to create report (or Cancel to exit)
