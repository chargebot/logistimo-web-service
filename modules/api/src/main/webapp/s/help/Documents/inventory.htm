Inventory

To view and manage inventory configuration, perform the following steps:

1.  Go to Configuration > Inventory

2.  Specify the transaction reasons, as briefed below:

•      Enter a coma-separated list of reasons you wish in the appropriate fields for transactions (this includes reasons for issues, receipts, stock count, discards and transfer transactions).

•      Ensure that the reason is small (less than 15 characters), such as damaged, expired etc.

3.  Specify permissions, by selecting inventory visibility if you want to allow inventory of customers to be visible to vendors during a transaction.

4.  Specify the export of transaction data, as briefed below:

•      Select enable data export of transactions in a CSV file, which is sent as an attachment to the specified Email ID.

•      Select the particular time.

•      Enter the specific Email ID you wish.

5.  Select the consumption rates manually from the drop down list.

•      To view measures either daily, weekly or monthly.

6.  Specify forecasting & optimization options, as briefed below:

•      Select the appropriate option for inventory forecasting and/or optimization and select relevant parameters (this includes none, automatic consumption rates, by demand forecasts or optimal order quantities)

7.  Click Save
