Create an entity

To create an entity, perform the following steps:

1.  Go to Setup > Entities.

2.  Click Add new Entity button.

3.  In the ensuing form, enter all the entity details.

4.  Specify Entity Details, as briefed below:

•      Enter Entity name.

•      Enter users name(s), get suggestions as you type and select the users name(s) as per necessity.

•      Optional Fields: Select Tag and if there is a Custom ID for this user, which you wish to map the user to, please enter that.

5.  Entity Location:

•      Enter Country and State in the appropriate fields.

•      Specify Village/City.

•      Optional fields: Can be entered by clicking the + sign near the title of this section (these includes District/ County,Taluk, Zip/PIN code, Street address and Geo Codes).

•      To map a location there are 3 ways

§ Type the location in village/city box ( like google search) and get specific suggestions

§ In can of multiple selections Click on the most accurate selection from the multiple selections

§ Pin could be dragged and dropped on a specific location

6.  Entity Financial Details:

•      Optional fields: Can be entered by clicking the + sign near the title of this section  (these include Currency, Tax (%) and Tax ID)

7.  Click Save to create new entity (or Cancel if you do not want to create the new account).
