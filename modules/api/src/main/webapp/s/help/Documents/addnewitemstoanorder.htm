Add new items to an order

To add new items in order, perform the following steps:

1.  Go to Orders > Orders

2.  Click order you wish to view.

3.  If you wish to add new items to an order in the ensuing order panel, click ‘Add’ link just above the items list.

4.  Add relevant item with quantity details

5.  Enter comment (if any)

6.  Click Save to update new items (or Cancel to exit)
