Add materials to an entity 

To add materials to an existing entity, perform the following steps:

1.  Go to Setup > Entities.

2.  Select the entity that you wish to edit materials on.

•      You can also search for an entity by entering entity name (including partial names) in the Search field and click Search button.

3.  In the ensuring entity profile page, click Materials at the top-right.

4.  In the ensuing material listing page, click Add new Material button.

5.  In the ensuing page, select the materials you wish to add by checking the box adjacent to the material name.

6.  Click Save.

7.  In the material listing page, click Refresh list to see the new materials on this entity.

 

Edit materials in an entity

To edit materials to an existing entity, perform the following steps:

1.  Go to Setup > Entities.

2.  Select the entity that you wish to edit materials on.

•      You can also search for an entity by entering entity name (including partial names) in the Search field and click Search button.

3.  In the ensuing entity profile page, click Materials at the top-right.

4.  In the ensuing page, select the materials you wish to edit by selecting the box adjacent to the material name.

5.  Click the Edit materials button.

6.  Click Save.

7.  In the material listing page, click Refresh list to see the edited material(s) on this entity.
