Upload materials in bulk

To upload materials in bulk, perform the following steps:

7.  Go to Setup > Materials.

8.  Click ‘Upload in bulk’ link at the top right of the domain.

9.  Click ‘Download data file format for materials’ link.

10. Once the file is downloaded, fill in the instructions as given in the column headers.

11. Upload the selected CSV file for materials

•      Click Choose File to upload the file.

12. Click Upload button to upload the file or Cancel to exit.

•      Successfully uploaded file will show a done status.

•      If the file consists of even one error, the error(s) will be shown and the file will not be uploaded.
