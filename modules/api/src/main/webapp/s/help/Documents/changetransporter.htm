Change transporter

To change transporter, perform the following steps:

1.  Go to Orders > Orders

2.  Click order you wish to edit.

3.  If you wish to change transporter in the ensuing order panel, click ‘Change’ link aligned right to transporter.

4.  Enter transporter

5.  Click Reset button, to set pack back to original size or

6.  Click Save to update transporter (or Cancel to exit)
