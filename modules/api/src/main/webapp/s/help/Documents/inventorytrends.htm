Inventory trends

To view reports by inventory trends, perform the following steps:

1.  Go to Reports

2.  Select Inventory trends in the Type search field.

3.  Click the From and To time fields.

•      Select the date range from the drop down calendar.

4.  Enter the mandatory materials filter field (which is in bold)

5.  Optional fields: like state, district/county, entity group, entity and entity tags) on which the report can be generated.

6.  Click the Get Report button

7.  The report will be rendered,

•      Select ‘Graphics’ to view of the report in the form of a graph or to view in tabular form select ‘Tables’

•      The overview shows the stock for that period, issues, receipts discards and closing stock as you require.

•      To view reports by stock, issues, receipts, discards and transfers, select the appropriate fields.
