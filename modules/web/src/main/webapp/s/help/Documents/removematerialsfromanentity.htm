Remove materials from an entity

To remove materials from an existing entity, perform the following steps:

1.  Go to Setup > Entities.

2.  Select the entity that you wish to edit materials on.

•      You can also search for an entity by entering entity name (including partial names) in the Search field and click Search button.

3.  In the ensuring entity profile page, click Materials at the top-right.

4.  In the ensuing material listing page, click the Remove materials button.

5.  In the ensuing page, select the materials you wish to add by checking the box adjacent to the material name.

6.  Click Save.

7.  In the material listing page, click Refresh list to see if the material(s) removed on this entity.
