Remove relationships from an entity. 

To remove relationships from an existing entity, perform the following steps: 

1.  Go to Setup > Entities. 

2.  Select the entity that you wish to remove relationships from.

•      You can also search for an entity by entering entity name (including partial names) in the Search field and click Search button.

3.  In the ensuing entity profile page, click Relationships at the top-right. 

4.  Select the relationship you want to remove.

5.  Click Remove relationship button.

6.  A pop-up box will appear for confirmation.

7.  Click OK if you wish to proceed (or Cancel to exit).
