Filter by status

To view orders by status, perform the following steps:

1.  Go to Orders > Orders

2.  Start typing the order status in the Status search field.

•      A list of order status will appear in the drop down list.

•      Select the status you wish from the list

i. pending: an order was placed and has not yet been processed.

ii.          confirmed: the order is accepted and under processing or preparation.

iii.         shipped: the order has been shipped.

iv.         fulfilled: the shipment has been received by the one who ordered.

v.         cancelled: the order has been cancelled.

2.  All orders of that status will appear.

3.  Click View button.

4.  If entity tags are shown, you can browse by entity tags.
