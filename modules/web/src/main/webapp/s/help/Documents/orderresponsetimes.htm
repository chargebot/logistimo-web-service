Order response times

To view reports by inventory trends, perform the following steps:

1.  Go to Reports

2.  Select Order response times in the Type search field.

3.  Click the From and To time fields.

a.  Select the date range from the drop down calendar.

4.  Optional fields: like Materials, material tags, state, district/county, entity group, entity and entity tags) on which the report can be generated.

5.  Click the Get Report button

6.  The report will be rendered,

•      Select ‘Graphics’ to view of the report in the form of a graph or to view in tabular form select ‘Tables’

•      By default the report in this domain is displayed ‘processing time’.

•      To view report in ‘Delivery lead time’, select from the filter in the top right.
