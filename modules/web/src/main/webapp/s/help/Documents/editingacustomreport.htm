Editing a custom report

To edit an custom report template, perform the following steps:

1.  Go to Configuration > Custom reports.

2.  Click on the ‘Report template name’ to edit it.

3.  Note: the template name cannot be edited.

4.  Any of the other fields can be edited as you wish.

5.  You can also upload a new file for an existing template

6.  Click Save to make changes to the custom report template (or Cancel to exit)
