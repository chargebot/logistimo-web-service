Replenishment response times

To view replenishment response times, perform the following steps:

1.  Go to Reports

2.  Select replenishment response times in the Type search field.

3.  Click the From and To time fields.

•      Select the date range from the drop down calendar.

4.  Optional fields: like Materials, material tags, state, district/county, entity group, entity and entity tags) on which the report can be generated.

5.  Click the Get Report button

6.  The report will be rendered,

•      Select ‘Graphics’ to view of the report in the form of a graph or to view in tabular form select ‘Tables’

•      By default reports in this domain is displayed in zero stock.

•      To view report that is <min or >max, select the appropriate filter in the top right.
