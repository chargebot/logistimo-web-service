View users

1.  Go to Setup > Users.

2.  Lists of users in the domain are shown.

3.  Click on any user ID you wish to view

•      You can also search for a user by entering his first name (including partial names) in the Search field at the top left of the user domain and click search.

4.  View the users details in the list

•      The name of the user, his role, location of the user, his contact number and last login details are shown.

•      If reconnected then last reconnected view is also available.
