View transactions

To view transactions, perform the following steps:

1.  Go to Inventory > Transactions

2.  All the transactions in this domain are shown with the most recent transaction on top.

•      In the view, each entry in this indicates an inventory operation done on a particular material at a particular entity. The quantity entered in the transaction, time at which it was entered and the user are indicated. The closing stock of the transaction is also shown.

•      If the transaction was geo coded. A map icon like this  is shown beside the user name, moving the mouse on icon shows the location at which the transaction was done. Move mouse on map pins to see the accurate location of geo coordinates

•      Click on map pin to get an expanded view of the location where the transaction is done.

3.  If tags are shown, you can browse transaction by material or entity tags.

4.  The transactions can be paginated across list by clicking on ‘next’ or ‘prev’ links.
