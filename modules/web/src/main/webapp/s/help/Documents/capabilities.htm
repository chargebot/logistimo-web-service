Capabilities

To view and manage capabilities based on deployment, perform the following steps:

1.  Go to Configuration > Capabilities

2.  Specify the role by clicking in the field. By default the roles of All is displayed in the select role field. You can tailor capabilities specific to the role, the roles are explained below:

•      Operator: An Operator has the ability to view and transact inventory and orders. But not manage entities, users and materials. He can only access logistimo using mobile and HTML application.

•      Manager: Manager of one or more entities, with ability to view, transact and manage (entity/users/materials)

•      Administrator: Owns and configures the domain, and can create and manage multiple entities, users (including Entity Managers and Operators), and materials.

3.  Specify the menu items that you wish to disable.

•      You can disable inventory by selecting either stock view, entering issues, receipts, stock counts, discards or stock transfers.

•      You can disable orders by selecting view stock orders and view orders.

4.  Select the types of entities to allow creation of entities on mobile (that includes entities, customers and vendors)

5.  Select remote login, if you wish to make ‘login’ to connect to server each time (reconnect/ refresh option will not appear on mobile)

6.  Enter the tags of materials you wish to be hidden from inventory view or order view.

7.  Select the related entity information to be sent on the mobile (that includes vendor and customer information)

8.  Select the Geo-location acquisition strategy you wish to view on the mobile

•      Optimistic- Has better performance, geo-location is acquired once during login and later on expiry.

•      Strict- At the cost of performance, geo-location is acquired just before sending transaction and has better freshness.

9.  Select the naming convention (to be used in naming transactions)

10. Click Save.
