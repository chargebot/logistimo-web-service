Editing a user profile

To edit user's account, perform the following steps:

1.  Go to Setup > Users.

2.  Click on the user ID you wish to edit.

•      You can also search for a user by entering his first name (including partial names) in the Search field and click Search button.

3.  At the bottom of the user's profile page, click Edit User button.

4.  Edit the required fields of the user.

5.  Click Save to retain the edited information of the user (or Cancel to discard the edited details).
