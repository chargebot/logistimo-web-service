View by type of abnormality

To view stock by type of abnormality, perform the following steps:

1.  Go to Inventory > Abnormal Stock

2.  By default all inventory that is zero stock is displayed along with the duration (in days) over which it is zero.

3.  This view can be browsed either by entity or material tags, if tags are available.

4.  To view inventory that is <min or >max. select the appropriate filter in the top right.

5.  Sort or group the rows in the view by column (point by clicking on the column header)
