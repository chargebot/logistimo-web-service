Upload users in bulk

To upload users in bulk, perform the following steps:

1.  Go to Setup > Users.

2.  Click ‘Upload in bulk’ link at the top right of the domain.

3.  Click ‘Download data file format for users’ link.

4.  Once the file is downloaded, fill in the instructions as given in the column headers.

5.  Upload the selected CSV file for users

•      Click Choose File to upload the file.

6.  Click Upload button to upload the file or Cancel to exit.

•      Successfully uploaded file will show a done status.

•      If the file consists of even one error, the error(s) will be shown and the file will not be uploaded.
