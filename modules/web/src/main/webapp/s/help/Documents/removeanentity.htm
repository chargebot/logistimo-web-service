Remove an entity

To remove an existing entity, perform the following steps:

1.  Go to Setup > Entities.

2.  Select the entity that you wish to remove entities on.

•      You can also search for an existing entity name (including partial names) in the Search field and click Search button.

3.  Click Remove Entities button.

4.  A pop-up box will appear for confirmation.

5.  Click OK if you wish to proceed (or Cancel to exit).
