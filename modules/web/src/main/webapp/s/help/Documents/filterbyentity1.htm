Filter by entity

To view transactions by entity, perform the following steps:

1.  Go to Inventory > Transactions

2.  Start typing the entity name in the Entity search field.

•      A list of entities will appear in the drop down list

•      Select the entity you wish from the list.

3.  Click View button.

4.  View the entity’s current transactions.

5.  The last updated view along with the user who last updated the transaction is also available.

6.  If material tags are shown, you can also browse by material tags.
