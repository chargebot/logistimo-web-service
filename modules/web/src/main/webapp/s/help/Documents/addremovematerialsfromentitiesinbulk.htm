Add/Remove materials from entities in bulk

To add/remove materials to an existing entity, by performing the following steps:

1.  Go to Setup > Entities.

2.  Select the entities that you wish to edit materials on.

3.  Click Add/Remove materials button.

•      If you do not select any entity. A pop-up box will appear for confirmation.

•      Click OK if you wish to proceed (or Cancel to exit)

4.  Check the box adjacent to the material's row you wish to add/remove.
