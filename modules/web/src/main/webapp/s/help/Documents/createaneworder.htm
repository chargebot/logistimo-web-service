Create a new order

To create a new order, perform the following steps:

1.  Go to Orders > Orders

5.  Start typing the entity name in the Entity search field.

•      A list of entities will appear in the drop down list

•      Select the entity you wish from the list.

2.  Click Add button.

3.  In the ensuing add order page,

•      Select the vendor, enter initial letters of the vendor and select the name

•      Optional fields: these include Status and Message

•      Enter quantity in the quantity fields.

4.  Click Submit button below the page, once you fill the form as required or Reset if you do not wish to make the changes you made in the form.
