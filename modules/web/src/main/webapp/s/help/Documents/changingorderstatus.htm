Changing order status

To change orders by status, perform the following steps:

1.  Go to Orders > Orders

2.  Click order you wish to change status on.

3.  In the ensuing order panel, you will see the status of the order on the top.

4.  To change the order of the status, click ‘Change’ link next to the order status

5.  Select the appropriate status.

6.  Status of an order are interpreted as stated below:

•      pending: an order was placed and has not yet been processed.

•      confirmed: the order is accepted and under processing or preparation.

•      shipped: the order has been shipped. Orders cannot be shipped if vendor stock is low.

•      fulfilled: the shipment has been received by the one who ordered.

•      cancelled: the order has been cancelled.

7.  Enter a comment (if any)

8.  Select the Send message via to... box if you wish to send a message, and the message will be sent via SMS to the person who placed the order.

9.  Click Save to update status (or Cancel to exit)
