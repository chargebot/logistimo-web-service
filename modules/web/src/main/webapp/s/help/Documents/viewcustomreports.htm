View custom reports

To view custom reports, perform the following steps:

1.  Go to Configuration > Custom reports.

2.  In this domain the report template name, report template file and last updated view is available.

3.  A delete icon  is also available, if you wish to delete the report.

4.  The report templates are excel files (.xls or .xlsx) that should be uploaded to the system.

5.  The exported data is pasted into the report periodically and emailed to the configured users.

6.  You can also click on the report template file to download and view the existing report.
